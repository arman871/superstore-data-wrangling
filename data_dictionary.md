# Data Dictionary

| Column | Type | Description |
|------|------|------|
| Row ID | Integer | Unique row identifier |
| Order ID | Text | Unique order number |
| Order Date | Date | Date when order was placed |
| Ship Date | Date | Date when order shipped |
| Ship Mode | Text | Shipping method |
| Customer ID | Text | Unique customer identifier |
| Customer Name | Text | Name of customer |
| Segment | Text | Customer segment |
| Country | Text | Customer country |
| City | Text | City of delivery |
| State | Text | State of customer |
| Region | Text | Sales region |
| Category | Text | Product category |
| Sub-Category | Text | Product sub category |
| Product Name | Text | Name of product |
| Sales | Float | Revenue generated |
| Quantity | Integer | Number of items sold |
| Discount | Float | Discount applied |
| Profit | Float | Profit earned |
| Delivery Days | Integer | Days between order and shipping |
| Profit Margin | Float | Profit divided by sales |
