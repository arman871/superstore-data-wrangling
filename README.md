# Superstore Data Wrangling

## Objective
Clean and prepare raw sales data for analysis.

## Dataset
Superstore sales dataset containing order, customer, and product information.

## Steps Performed
- Data exploration
- Data quality assessment
- Removing duplicates
- Converting date formats
- Feature engineering

## New Features

- Profit Margin
- Sales Category
